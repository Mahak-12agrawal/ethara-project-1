# Student Management System 🎓

A modern and responsive **Student Management System** built using **React.js**, **Tailwind CSS**, **Node.js**, **Express.js**, and **MongoDB**.

This project helps schools, colleges, and educational institutes efficiently manage student records through a clean dashboard interface with CRUD operations, responsive UI, search functionality, and scalable architecture.

---

✅ Add Student Records
✅ Update Student Information

# ⚙️ Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/Mahak-12agrawal/student-management-system.git
```

---

## 2️⃣ Navigate to Project

```bash
cd student-management-system
```

---

# Frontend Setup 🚀

## 3️⃣ Open Client Folder

```bash
cd client
```

---

## 4️⃣ Install Dependencies

```bash
npm install
```

---

## 5️⃣ Start Frontend Server

```bash
npm run dev
```

Frontend runs on:

```bash
http://localhost:5173
```

---

# Backend Setup 🔥

## 6️⃣ Open Server Folder

```bash
cd server
```

---

## 7️⃣ Install Backend Packages

```bash
npm install express cors mongoose dotenv
```

---

## 8️⃣ Start Backend Server

```bash
node server.js
```

Backend runs on:

```bash
http://localhost:5000
```
